### 💡 商业价值 / Business Value
- **[CN]** 专治企业多系统异构（POS/客流/财务）带来的数据内耗与合规风控漏洞。利用自动化脚本打通数据孤岛，实现千万级复杂账务的“零差错”精细化对账，并利用 AI 智能体将非结构化条款与结构化账单自动关联，用机器代替人工进行每日数据追踪与危机拦截。
- **[EN]** Engineered to eliminate corporate data friction and compliance vulnerabilities caused by disjointed, multi-system environments (POS/Traffic/Finance). Leverages automated data pipelines to achieve "zero-error" reconciliation across millions of transaction rows, utilizing low-code AI agents to link unstructured contract clauses with structured ledgers for autonomous risk alerting.

---

### 🛠️ 核心功能模块 / Core Modules
- **🧹 自动化数据清洗与建模 / Automated ETL Pipeline**: Use Python to automatically clean messy raw exports from multiple separate databases, modeling a clean, standardized tenant-date-sales fact table.
- **📈 动态经营驾驶舱 / Business Intelligence Dashboard**: Built dynamic dashboards via Power BI/Streamlit featuring floor-level sales heatmaps, tenant rent-to-sales ratios, and payment anomaly monitors with daily auto-refresh.
- **🤖 AI 对账与群预警智能体 / AI Agent Auditing & Alerting**: Configured a specialized agent in Dify/Coze that auto-checks non-structured contract terms against structural ledgers, instantly pushing anomalies and expiration alerts into simulated management communication channels.

---

### ⚙️ 核心技能栈 / Capabilities Demonstrated
- **SQL / Python**: Advanced multi-table JOINs, GROUP BY optimizations, and raw JSON parsing.
- **BI & Low-Code AI**: Power BI, Streamlit Web App, Dify/Coze Agent Orchestration.
